## Task 1: Population Distribution Visualization

### Objective
Create a bar chart to visualize the distribution of a categorical variable (Gender) using Python.

### Tools & Libraries
- Python
- Pandas
- Matplotlib

### Dataset
- Gender Population Dataset (CSV)

### Steps Performed
1. Loaded the dataset using Pandas.
2. Counted the number of Male and Female records.
3. Created a bar chart using Matplotlib.
4. Saved the chart as a PNG image.
